Sobre mí

¿Quién es David?

Desarrollador Full-Stack en ciernes.

Me llamo David y soy el tipo de desarrollador que construye tanto el escenario como el detrás de cámaras. Desde diseñar interfaces atractivas en el frontend, hasta estructurar bases de datos robustas en el backend, disfruto cada línea de código que conecta la experiencia del usuario con la lógica del servidor.

Después de cursar mis estudios sanitarios, me dí cuenta de que no era para mi, y decidí dedicarme a lo que realmente siempre me había gustado: la programación.

Fullstack por convicción, curioso e investigador por naturaleza. Me encanta resolver problemas, optimizar procesos y aprender nuevas tecnologías que me hagan mejor cada día. Si algo puede automatizarse, mejor. Y si puede hacerse con estilo, aún mejor!

Proyectos

Biblioteca Social (red social para lectores)

Red social sencilla donde los usuarios pueden registrarse y subir sus lecturas, tanto finalizadas como pendientes

Contiene un sistema de amistad para que los usuarios puedan interactuar entre sí.

Realizada en PHP y Laravel en el lado servidor y HTML, CSS y JavaScript en el lado cliente. Incluye integración con base de datos y uso de APIs para la búsqueda automática de libros

Ver código en GitHub Repo

Proyecto invitados para boda

Aplicación web desarrollada por interés propio donde el usuario puede llevar la cuenta de los invitados a su boda.

Diferencia entre invitados del novio y de la novia, los que confirman, los que dicen que no y los que están dudosos. Diferencia entre adultos, acompañantes y niños.

Contiene contadores para saber en todo momento cuantos invitados hay confirmados y cuantos no.

Aplicación desarrollada con PHP y Laravel en el lado servidor y HTML, CSS y JavaScript en el lado cliente. Incluye integración con base de datos.

Ver código en GitHub Repo

Plataforma de gestión para empresa de servicios informáticos

Plataforma de gestión para empresa de servicios informáticos para tener un control más sencillo sobre todos los datos de sus clientes y servicios.

Entre estos apartados se encuentran la gestión de usuarios (con roles como usuario y administrador), dominios, cuentas, hosting, renovaciones, mantenimientos, licencias, planes, etc.

Contiene funciones como obtener los NS automáticamente, exportar a Excel el calendario o importar archivos, entre otros.

Desarrollada con PHP y Laravel en el lado servidor y HTML, CSS y JavaScript en el lado cliente. Incluye integración con base de datos.

Ver código en GitHub Repo
